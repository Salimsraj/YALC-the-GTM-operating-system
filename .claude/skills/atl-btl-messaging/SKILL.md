# ATL vs BTL Messaging

Use this skill to match the message to the target seniority.

## When This Skill Applies

- Writing for VP/C-level versus manager/IC audiences
- Adjusting message length, tone, and proof points
- Segmenting campaigns by buying role

## What It Does

1. Chooses the correct seniority lane.
2. Sets tone and detail level for that lane.
3. Produces a message shape that is easier to personalize.

## Pair With YALC

- Use `personalize` to write the final text.
- Use `linkedin:answer-comments` or `linkedin:reply-to-comments` for LinkedIn execution.
