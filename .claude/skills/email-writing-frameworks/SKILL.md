# Email Writing Frameworks

Use this skill to choose the structure of a cold email.

## When This Skill Applies

- Writing Email 1 from scratch
- Reworking an underperforming message
- Picking a framework for a specific persona or offer

## What It Does

1. Picks the best framework for the target.
2. Converts the framework into a short email body.
3. Keeps the copy tight, direct, and testable.

## Pair With YALC

- Use `personalize` for the actual message drafting.
- Use `send-cold-email` once the framework is approved.
