# Lead Sources Guide

Use this skill to choose the best discovery source for a prospecting motion.

## When This Skill Applies

- Deciding which data source to use first
- Building a sourcing workflow for a new campaign
- Comparing platform strengths for discovery and enrichment

## What It Does

1. Matches the use case to the right source.
2. Suggests the next step after discovery.
3. Keeps the workflow focused on actionable data.

## Pair With YALC

- Use `provider:list` and `adapters:list` to see what is already connected.
- Use `leads:import` after you choose a source.
