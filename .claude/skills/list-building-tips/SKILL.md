# List Building Tips

Use this skill to keep list-building fast, clean, and repeatable.

## When This Skill Applies

- Building a new campaign list
- Mixing multiple sources into one set
- Improving list quality before qualification or export

## What It Does

1. Recommends a clean source order.
2. Prevents duplicate and low-quality records from spreading.
3. Keeps the list ready for downstream qualification.

## Pair With YALC

- Use `leads:import` to load the list.
- Use `leads:dedup` and `leads:qualify` before any send.
