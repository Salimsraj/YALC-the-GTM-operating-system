# Outbound Triggers

Use this skill to anchor pure cold outreach on a business premise.

## When This Skill Applies

- Building top-down or bottom-up outbound campaigns
- Multi-threading into accounts
- Starting from a business problem instead of an event

## Pair With YALC

- Use `campaign:strategy` before creating the sequence.
- Use `campaign:create-sequence` or `email:create-sequence` to execute the motion.
